Landing Page Clair Obscur

HTML | CSS | JavaScript

Este projeto é uma landing page sobre no jogo Clair Obscur: Expedition 33.
Criei-a de forma descontraída, principalmente para praticar técnicas de desenvolvimento front-end, e também porque me apaixonei pelo visual e pela atmosfera do jogo.

O projeto foi desenvolvido utilizando apenas HTML, CSS e JavaScript puros, sem o uso de frameworks, com foco em aprimorar minhas habilidades de estruturação, estilização e interação básica com o usuário.

Além de ser um exercício técnico, também foi uma forma de unir duas coisas que gosto muito: design de interfaces e jogos.
